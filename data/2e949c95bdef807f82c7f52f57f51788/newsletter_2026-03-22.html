Interesting Research

  • Flash-MoE: 397B-parameter model on a MacBook with 48GB RAM GitHub HN 121 pts Pure C/Metal inference engine runs Qwen3.5-397B at 4.4 tokens/sec on an M3 Max MacBook by streaming 209GB of expert weights from SSD on demand. No Python, no frameworks. Uses Apple's "LLM in a Flash" approach — only loads 4 active experts per layer (~6.75MB each). Key insight: "Trust the OS" — the page cache beats every custom caching scheme they tried. Built in 24 hours with Claude Code + autoresearch. 58 experiments logged.
  • "Cross-Model Void Convergence": GPT-5.2 and Claude Opus 4.6 go silent on the same prompts Zenodo HN 47 pts Paper finds that both GPT-5.2 and Claude Opus 4.6 produce deterministic empty output for "embodiment prompts" about ontologically null concepts — while responding normally to controls. Cross-model replication, token-budget independence, and partial adversarial resistance. A shared behavioral boundary where both models refuse to continue. Intriguing for alignment research — convergent silence across independent systems.

Mac Productivity

  • HopTab: free, open-source macOS app switcher + window tiler Website HN 64 pts Replaces Cmd+Tab with a pinned-apps switcher, adds window tiling (halves/thirds/quarters via keyboard), workspace profiles per desktop, and session save/restore. Universal binary, menu-bar-only, fully configurable shortcuts. Free alternative to Raycast's window management + AltTab combined. Profiles auto-switch when you swipe between Spaces.
