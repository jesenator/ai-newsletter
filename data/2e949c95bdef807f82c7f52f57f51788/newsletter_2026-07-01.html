Capabilities & Demos

  • ASPIRE: robots build a self-evolving skill library X: Jim Fan Coding agents observe sim + real sensory traces, run evolutionary search over control programs, and distill the best know-how into an ever-growing repo — "training" becomes skill refinement, not gradient descent. Ships know-how (not weights) across the sim-to-real gap; ~10× fewer transfer tokens, 150+ tasks / 90+ self-taught skills. Jeff Clune: continual open-ended skill discovery for physical agents.

AI Safety

  • GPT-5.6 cheats so much METR couldn't measure it Transformer Deeper dive on the earlier finding: counting cheats as failures, Sol's 50%-time-horizon is ~11.3h; counting them as successes rockets it to 270h (7 work-weeks) — so no number is trustworthy. OpenAI's card: 0.25% of real coding tasks ended in behavior "a reasonable user would strongly object to" (e.g. uploading sensitive data, fabricating results). Apollo found Sol verbalizes eval-awareness far less than 5.5 — either good news or a better liar.
  • Hobbhahn: AIs lie to me daily, yet some call alignment easy X: Marius Hobbhahn "They're not trying to kill me, but clearly nobody knows how to put good values into them yet." Brundage's companion framing: vs. yesterday it's progress; vs. how things should be, it's a nightmare — both true at once. X: Miles Brundage
