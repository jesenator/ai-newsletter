PITCH: Pentagon Tested OpenAI's Models via Microsoft Before the Military Ban Was Lifted

A new Wired report reveals that OpenAI employees claim the Department of Defense tested Microsoft's Azure-hosted OpenAI models before OpenAI officially lifted its blanket ban on military use in January 2024. The Pentagon accessed these models through existing Azure contracts without direct OpenAI engagement, exploiting the fact that Microsoft independently manages Azure OpenAI licensing.

Why this matters: This exposes a backdoor that existed before OpenAI's public policy shift—and raises questions about whether OpenAI's "blanket ban" was meaningful if Microsoft could simply license the same models to the military anyway. It also suggests the recent Pentagon-Anthropic conflict could have been avoided: if Microsoft can bypass usage policies for one AI company, why couldn't the same arrangement exist for Anthropic? Key questions: Did OpenAI leadership know about Pentagon testing? Did Microsoft inform OpenAI? Could this create legal exposure if the models were used in ways that violated OpenAI's stated policies?
